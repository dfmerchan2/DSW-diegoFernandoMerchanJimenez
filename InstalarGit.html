Instalación de git

Instalar Git en Windows

1. Descargar el instalador de Git mas reciente

2. Ejecutar el instalador de Git




3. Abra git bash y ejecute el comandogit --version para conocer la version de Git




Instalar Git en Linux

1. Primero abrimos un terminal

sudo apt-get update

sudo apt-get upgrade

2. Ahora instalamos Git

sudo apt-get install git

3. Introduce el siguiente comando para verificar que la instalación se ha realizado correctamente

git --version git

4. Configuramos nuestro nombre de usuario

git config --global user.name "usuario"

5. Configuramos nuestra dirección de e-mail

git config --global user.email "usuario@email.com"
